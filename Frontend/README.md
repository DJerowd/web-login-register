## Techs

- [x] React.js + Vite
- [x] JavaScript
- [x] Axios
- [x] React Hook Form
- [x] React Router Dom

## How to download the project

```bash
    # Clone the repository
    $ git clone https://github.com/DJerowd/web-login-register.git

    # Enter directory
    $ cd web-login-register
```

## Usage

1. Run `npm install`.<br />
2. Run 'npm run dev' and access http://localhost:5173/<br />
